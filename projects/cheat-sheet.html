Basic SQL Cheat Sheet

UPDATE

-edits a row in a table

Syntax

UPDATE table_name
SET column1 = value1, column2 = value2, ect...
WHERE some_column = some_value;

Example

UPDATE customers
SET name = "George", city = "chicago", country = "USA"
WHERE name = "George";

Notes

Using WHERE is important so you don't edit every row in the database

ALTER TABLE

-cahnges an existing table

Syntax

ALTER TABLE table_name
ADD column_name datatype;

Example

ALTER TABLE customers
ADD birthday date;

Notes

In place of 'ADD' you can use many different commands including 'MODIFY' to change an existing column, 'DROP' to drop an existing column or 'RENAME' to rename an existing column

DELETE FROM

-deletes a row from a table

Syntax

DELETE FROM table_name
WHERE some_column = some_value;

Example

DELETE FROM customers
WHERE name = 'George" AND name = 'Richard';

SELECT DISTINCT

-used to return unique values in the result set-- it filters out duplicate values

Syntax

SELECT DISTINCT column_name FROM table_name;

Example

SELECT DISTINCT genre FROM movies;
(this would list each genre in movies exactly once.)

Notes

WHERE

-used to filter queries

Syntax/Example

SELECT * FROM movies
WHERE imdb_rating > 8;

LIKE

-operator that gives you similar values

Syntax/Example

SELECT * FROM movies
WHERE name LIKE 'se_en';

Notes

The '_' is a wildcard character for one character

%

-another wildcard cahracter that matches 0 or more missing characters in a pattern

Syntax/Example

SELECT * FROM movies
WHERE name LIKE 'harry potter%";

Notes

This would return anything that starts with 'harry potter', it doesn't matter what comes after. (ideally it would return all the sequels!)

BETWEEN

-filters the result set within a certain range

Syntax

SELECT * FROM some_table
WHERE column_name BETWEEN parameter_1 AND parameter_2;

Example

SELECT * FROM movies
WHERE year BETWEEN 1990 AND 2000;
(or)
WHERE name BETWEEN 'A' AND 'J';

AND

-used to require multiple conditions

Syntax/Example

(see above example)

OR

-used to combine multiple conditions

Syntax/Example

SELECT * FROM movies
WHERE genre = 'comedy'
OR year < 1980;

ORDER BY

-sorts the results of queries

Syntax

SELECT * FROM table_name
ORDER BY some_column_value direction;

Example

SELECT * FROM movies
ORDER BY imdb_rating DESC;

Notes

DESC = descending order
ASC = ascending order

LIMIT

-specify the number of rows to be displayed

Syntax

SELECT * FROM some_table
WHERE column SOME PARAMETER
LIMIT some_number;

Example

SELECT * FROM movies
WHERE year BETWEEN 1990 AND 2000
LIMIT 5;
